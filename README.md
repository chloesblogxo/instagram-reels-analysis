# instagram-reels-analysis
Statistical analysis of Instagram Reel retention, skip rates, and follower conversion using R. 

Beyond Views: A Statistical Analysis of Instagram Reel Retention and Audience Growth

Overview:
This project analyzes 50 Instagram Reels posted on my creator account to investigate which creative decisions are associated with viewer retention and follower growth.

Using R, I collected and analyzed metrics including:
- Hook type
- First-frame composition
- Caption characteristics
- Reel length
- Skip rate
- Follower conversion

The goal was to identify which content characteristics are associated with stronger performance for creators and brands.

Research Questions:
- Which hook types are associated with lower skip rates?
- Does first-frame composition influence viewer retention?
- Does reel length affect skip rate?
- Which variables best predict follower growth?

Methods: 
- Exploratory Data Analysis
- Linear Regression
- Correlation Analysis
- Data Cleaning
- Visualization using ggplot2

Key Findings:
- Original creative hooks and trend-based hooks produced lower skip rates.
- Face-first opening shots outperformed product-focused openings.
- Reel length showed only a weak relationship with skip rate.
- The highest-viewed hooks were not necessarily the strongest predictors of follower growth.
